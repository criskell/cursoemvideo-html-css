Principais formatações

Texto em itálico / Texto com ênfase

A tag i é utilizada para deixar um texto em itálico. Esta tag não é semântica pois está focada mais na forma do que no significado do conteúdo.

A tag em (emphasis) é utilizada para dar ênfase (significado) a um conteúdo. Esta tag é semântica pois está mais focada no significado do que na forma do conteúdo.

Texto em negrito / Texto em destaque

A tag b é utilizada para deixar um texto em negrito. Esta tag não é semântica pois está focada mais na forma do conteúdo do que em seu significado.

A tag strong é utilizada para destacar um texto. Esta tag é semântica pois está focada mais em dar significado (o destaque para o texto) ao invés de focar em sua forma.

Texto marcado

Podemos marcar texto utilizando a tag mark.

Outro texto marcado.

Mais um texto marcado.

Texto grande e texto pequeno

Podemos escrever textos grandes utilizando a tag big e textos pequenos utilizando a tag small. A tag big agora está obsoleta.

Textos deletado

Podemos utilizarmos a tag del para escrever um texto que deve ser lido mas desconsiderado.

Este texto deve ser lido, mas desconsiderado.

Texto inserido

Utilizamos a tag ins para indicar que um texto deve ter ênfase e indicar que ele foi inserido depois.

Eu adicionei este texto depois e este texto tem uma ênfase.

Texto sobrescrito

A tag sup é utilizada para criar textos sobrescritos, por exemplo, xx2 + 20

Outro exemplo: xxxxxxxxxxxxx

Texto subscrito

A tag sub é utilizada para criar textos subscritos, por exemplo, A 1, A2, ...An.

Outro exemplo: AAAAAAAAAAAAAAAAA
