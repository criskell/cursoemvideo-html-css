Glossário de Python

Funções
Bloco de código nomeado que é executado de acordo com argumentos e retorna um resultado.
Variável
Nome que se refere a um valor.
Objeto
Abstração para dados. Possui 3 propriedades: identidade, tipo e conteúdo.
Argumento nomeado
Argumento que é passado através do nome do parâmetro.
Instrução
Unidade de código que o Python executa.
Programa
Uma sequência de instruções.

Glossário de HTML

HTML
HTML (HyperText Markup Language) é uma linguagem de marcação para a criação de páginas web.
Elemento
Pedaço de uma página web. Por exemplo, botões, seções, formulários e etc.
Tag
Tags são utilizadas para a criação de elementos.
Semântica
Semântica no HTML é o significado por oposição a sua forma. O HTML5 é mais focado em semântica.
Hierarquia de títulos
Hierarquia de títulos são os níveis (1-6) que um título pode ter numa página.
Texto deletado
Texto deletado é um texto envolvido pelas tags del que significa que o texto deve ser desconsiderado.
Texto inserido
Texto inserido é um texto envolvido pelas tags ins que significa o texto deve ser lido com atenção e/ou foi adicionado depois.
Abreviações
Abreviações são formas reduzidas de um texto e são criadas com a tag abbr.
